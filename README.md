A simple offline notebook app for Windows to write notes, draw sketches, manage todos, and organize daily work in one place.
It is designed for students and office users who prefer a private, local-first productivity tool without relying on the internet or cloud services.

This project started as a personal tool to manage my own study notes and daily tasks. Later, I felt it could be useful for others too, so I decided to publish it.

**What You Can Do With This App**

Write and format notes (bold, italic, headings, lists)

Use a whiteboard to scribble, draw diagrams, or take handwritten notes

Maintain a todo list for each notebook

Use it as a personal diary or daily journal

Automatic spell check while typing

Password-protect notebooks and lock the app

Backup and restore all data

Export notes and sketches to PDF

Fast local search across notes

Works completely offline

Why This App?

No internet required

No cloud or account signup

Your data stays on your computer

Simple and distraction-free

Useful for study, office work, journaling, and idea capture
